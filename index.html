3AM-Tunes

3AM-Tunes is a Discord Music Bot developed by your average Discord user Sleep otherwise known as Zz or IsTired and has a web of himself here.
But enough about the developer, (techy stuff here so read if you want) 3AM-Tunes uses node.js so it can use disocrd.js and distube (which are both great) to run
and can use spotify links using spotify API, and on top of all that it uses Daki as a host (also amazing)
